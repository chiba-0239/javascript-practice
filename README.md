## 作品① 税込計算ツール

### Version1.3

### 機能

* 商品価格を入力
* 消費税率を入力
* 税込価格を計算
* 条件による送料判定

  * 1000円以上：送料無料
  * 500円以上：送料300円
  * 500円未満：送料500円
* 数字以外の入力をチェック
