# 作品① 税込計算ツール

## Version1.3

### 機能

* 商品価格を入力
* 消費税率を入力
* 税込価格を計算
* 条件による送料判定

  * 1000円以上：送料無料
  * 500円以上：送料300円
  * 500円未満：送料500円
* 数字以外の入力をチェック

### 学んだこと

入力チェックを実装し、数値以外が入力された場合の処理を追加しました。

また、消費税率を入力できるように改良する過程で、自分でバグを発見・修正し、デバッグの基本的な流れを経験しました。

---

## Version1.6

### 追加した機能

* 会員割引（5%）機能を追加
* 税込価格計算・会員割引・送料計算をそれぞれ関数へ分割
* 引数と戻り値を利用して関数同士でデータを受け渡す構成へ改善
* コメントや変数名を整理し、読みやすいコードへ改善

### 使用した技術

* function
* 引数
* return
* toFixed()
* 関数分割
* 条件分岐（if / else）

### 工夫した点

* 「1つの関数は1つの役割」を意識して設計しました。
* `calculateTax`、`applyMemberDiscount`、`calculateShipping` の3つの関数に役割を分割し、保守しやすい構成を目指しました。
* 関数の戻り値を次の関数へ渡すことで、処理の流れが分かりやすくなるように設計しました。
* `total`、`finalTotal`、`shipping` など、役割が分かる変数名を使用しました。

### 学んだこと

今回の作品制作を通して、コードを書くことだけでなく、処理を設計し、関数へ分割し、引数と戻り値を利用して処理を組み立てる考え方を学びました。

また、動作するだけでなく、他の人が読んでも理解しやすいコードを書くことの重要性を学びました。
